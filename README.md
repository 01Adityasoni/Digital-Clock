# Digital Clock

A simple and elegant digital clock web application built with HTML, CSS, and JavaScript.

## Features
- Real-time display of current hours, minutes, and seconds
- Responsive and modern design
- Easy to use and lightweight



## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)

### How to Run
1. Clone or download this repository to your local machine.
2. Open `index.html` in your web browser.

## Project Structure
```
index.html      # Main HTML file
styles.css      # CSS styles for the clock
script.js       # JavaScript logic for updating the time
README.md       # Project documentation
```

## Customization
- You can modify `styles.css` to change the appearance of the clock.
- Update `script.js` to add more features (e.g., 12/24-hour toggle, date display).
